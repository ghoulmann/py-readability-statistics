Readability Statistics

To analyze text readability, choose a plain text (.txt) file using the "browse" button.

When you've finished, press the "analyze" to have the specified text analyzed.

After pressing "Analyze," please be patient; each word in the selected text must be compared to a list of 3,000 words.
